# Cutline

Apple Silicon Mac上のAbleton Liveで、不要な低域や高域を取り除くVST3オーディオプラグインの開発を目的とするプロジェクトである。

## 開発状況

現在は仕様と設計判断を整理した段階である。実装コード、ビルド定義、テストはまだ含まれていないため、プラグインのビルドや使用はできない。
